How to get free Gemini key
  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the key — it's completely free!
How to get Claude API key

  2. Sign up or log in
  3. Go to API Keys section
  4. Click "Create Key" and copy it

🗄️ Supabase Database

Free cloud database to store your meals across devices.

How to set up Supabase
  1. Go to supabase.com and create a free account
  2. Click "New Project", pick a name and password
  3. Once created, go to Settings → API
  4. Copy your Project URL and anon public key
  5. Go to SQL Editor and run this SQL:
create table meals ( id uuid default gen_random_uuid() primary key, user_id text not null, date date not null, food_name text not null, portion text, calories integer, protein numeric, carbs numeric, fat numeric, fiber numeric, created_at timestamp default now() ); create table user_settings ( user_id text primary key, goals text, diet_mode text default 'balanced', updated_at timestamp default now() ); create table activities ( id uuid default gen_random_uuid() primary key, user_id text not null, date date not null, activity_name text not null, duration_min integer default 0, calories_burned integer default 0, created_at timestamp default now() ); -- Run this if you already have the tables above: alter table user_settings add column if not exists body_stats text; create table weight_logs ( id uuid default gen_random_uuid() primary key, user_id text not null, date date not null, weight numeric not null, created_at timestamp default now(), unique(user_id, date) );

All keys are stored locally in your browser. Nothing is sent to any server except the respective APIs.
